# Brain Games
Brain games is set of five small logical games including
* brain-even
* brain-calc
* brain-gcd
* brain-prime
* brain-progression
# Installing
make install
# Run games
* make brain-even
* make brain-calc
* make brain-gcd
* make brain-prime
* make brain-progression
